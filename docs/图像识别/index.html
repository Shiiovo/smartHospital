皮肤病识别

基本信息​

项目说明
接口URLhttp://114.132.235.143:8002/identify
请求方法POST
数据格式multipart/form-data
支持跨域是

请求参数​

接口支持两种方式提交图片，二选一即可。

参数名类型必选描述
fileFile否皮肤病图片文件（支持jpg, jpeg, png等常见格式）
image_urlString否皮肤病图片的URL地址（必须是直接图片链接）

响应格式​

成功响应

{
  "prediction": "皮肤病类型名称"
}

失败响应

{
  "error": "错误描述信息"
}

支持的皮肤病类型​

API可以识别以下8种常见皮肤病：

  • 白癜风
  • 斑疹
  • 带状疱疹
  • 皮肤过敏
  • 湿疹
  • 荨麻疹
  • 痤疮
  • 真菌感染

状态码说明​

状态码说明
200请求成功
400请求参数错误
500服务器内部错误

使用注意事项​

  • 图片应清晰显示皮肤病变区域
  • 提供的图片URL必须是直接指向图片的链接，而非搜索页面或图片展示页面
  • 支持的图片格式包括JPG、JPEG、PNG等常见格式
  • 建议图片大小不超过5MB
  • 接口已配置CORS，支持跨域请求
